# AdSense Resubmission Runbook (2026-02-25)

대상 사유: `가치가 별로 없는 콘텐츠 (Low value content)`

## 이번 반영 내용 (완료)
- `JS 템플릿 상세 페이지` 중심 구조 → `정적 상세 페이지 (/places/*.html)` 추가
- 상세 페이지 130개 생성 (본문/FAQ/체크리스트/내부링크 포함)
- `place.html` 템플릿 `noindex,follow` 처리
- 내부 링크 및 sitemap을 `/places/*.html` 중심으로 전환
- 상위 20개 장소 수기 편집 메모 추가

## 재심사 전 체크리스트 (필수)
- `about`, `editorial`, `privacy`, `terms`, `partner` 정상 노출
- 광고 코드가 과도하게 상단/본문을 가리지 않음
- 빈 페이지/공사중 페이지 없음
- 모바일에서 본문이 잘 읽힘 (가독성, 줄간격, CTA 과밀 여부)
- 404/오류 링크 최소화
- `place.html`이 검색 결과에 노출되지 않도록 `noindex` 유지

## Google 정책 링크 기반 점검 포인트
- 최소 콘텐츠 요건: 고유 본문, 사용자에게 유용한 정보, 탐색성
- 얇은 콘텐츠 회피: 자동 생성처럼 보이는 반복 문장 비중 축소
- 부가 가치 제공: 단순 링크 모음이 아니라 실제 방문 판단에 도움이 되는 팁 제공
- 신뢰성: 운영 원칙/정책 페이지/문의 경로 노출

## 재심사 실행 타이밍 (권장)
1. 배포 완료
2. Search Console에서 `/places/*.html` 우선 URL 색인 요청
3. 최소 5~10일 반영 모니터링
4. 색인 확인 후 AdSense 재심사 요청

## 모니터링 항목 (재심사 전)
- 유입/페이지뷰가 `/places/*.html`로 발생하는지
- Search Console에서 색인된 페이지 수 증가
- 중복 URL(`place?id=`)의 노출 감소
- 크롤링 오류/soft 404 없음

## 재심사 요청 시 메모(직접 입력용)
- "정적 상세 페이지를 추가하고, 기존 템플릿 상세 페이지는 noindex 처리했습니다."
- "고유 본문/FAQ/여행 계획용 안내를 상세 페이지에 추가했습니다."
- "내부 링크 및 sitemap을 색인 가능한 상세 페이지 기준으로 정리했습니다."
